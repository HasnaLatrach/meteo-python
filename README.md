# Outil de Prévision Météorologique Personnalisée

## Présentation du projet

Ce projet est une application web de prévision météorologique personnalisée, réalisée dans le cadre du module **Compétences Numériques et Informatique**.

L’application permet de consulter des prévisions météo, de visualiser les données sous forme de graphiques, de comparer plusieurs villes et d’exporter les résultats dans différents formats.

## Fonctionnalités principales

- Consultation des prévisions météorologiques actuelles
- Affichage des prévisions horaires et quotidiennes
- Visualisation des données météo avec des graphiques interactifs
- Analyse statistique des données météorologiques
- Comparaison de plusieurs villes
- Calcul d’un score de confort météorologique
- Exportation des données en CSV, JSON et PDF

## Technologies utilisées

- Python
- Streamlit
- Open-Meteo API
- Pandas
- Plotly
- Requests
- ReportLab

L’objectif de ce projet est de proposer une application simple, interactive et utile permettant d’exploiter les données météorologiques de manière claire et personnalisée.
